github创建个人仓库

首先登陆github账号，点击 GitHub 中的 New repository 创建新仓库，仓库名应该为：用户名 .github.io 。这里用户名使用你的 GitHub 帐号名称代替，这是固定写法。
注意：将仓库属性设置为public，公共仓库。
